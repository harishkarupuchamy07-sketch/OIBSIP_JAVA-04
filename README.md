# 🏧 ATM Management System

## 📌 Project Overview

ATM Management System is a console-based banking application developed using Java. The system allows users to securely log in and perform banking operations such as deposits, withdrawals, fund transfers, and viewing transaction history.

## 🎯 Features

* User Authentication (User ID & PIN)
* Deposit Money
* Withdraw Money
* Transfer Funds
* Transaction History
* Balance Management
* Menu-Driven Interface

## 🛠 Technologies Used

* Java
* ArrayList
* Scanner Class
* Object-Oriented Programming Concepts

## 🚀 How to Run

### Compile the Program

```bash
javac ATMSystem.java
```

### Run the Program

```bash
java ATMSystem
```

| User ID | Karthiga |

## 📋 Available Operations

### 1. Transaction History

Displays all previous transactions.

### 2. Withdraw

Allows users to withdraw money if sufficient balance is available.

### 3. Deposit

Allows users to add money to their account.

### 4. Transfer

Allows users to transfer funds from their account.

### 5. Quit

Exits the application safely.
