# 🏧 Java ATM System

A simple ATM System developed using Java that simulates basic banking operations such as login authentication, deposit, withdrawal, fund transfer, and transaction history management.

## 📌 Features

- User Login Authentication (User ID & PIN)
- View Transaction History
- Withdraw Money
- Deposit Money
- Transfer Funds
- Balance Management
- Menu-Driven Console Interface

## 🛠 Technologies Used

- Java
- ArrayList
- Scanner Class
- Object-Oriented Programming Concepts

## 🚀 How to Run

1. Clone the repository:

```bash
git clone https://github.com/yourusername/java-atm-system.git
```

2. Open the project in your preferred IDE (VS Code, IntelliJ IDEA, Eclipse).

3. Compile the program:

```bash
javac ATMSystem.java
```

4. Run the program:

```bash
java ATMSystem
```

## 📋 ATM Operations

### 1. Transaction History
Displays all previous transactions performed by the user.

### 2. Withdraw
Allows users to withdraw money if sufficient balance is available.

### 3. Deposit
Allows users to add money to their account.

### 4. Transfer
Allows users to transfer funds from their account.

### 5. Quit
Exits the ATM application safely.
